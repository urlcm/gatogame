## 📌 Descripción
Aplicación de escritorio que implementa el clásico juego del **Gato (Tres en Raya)**, desarrollada en **C# con Windows Forms**, incorporando un oponente controlado por inteligencia artificial mediante el algoritmo **Minimax**.

El objetivo del proyecto es demostrar la implementación de algoritmos de toma de decisiones en juegos, garantizando que la IA juegue de manera óptima.

## 🧠 Inteligencia Artificial

El juego utiliza el algoritmo **Minimax**, el cual permite a la IA:

- Evaluar todos los posibles movimientos futuros  
- Maximizar sus posibilidades de ganar  
- Minimizar las oportunidades del oponente  
- Tomar decisiones óptimas en cada turno  

👉 Esto hace que la IA sea **invencible** (si el jugador comete errores).
